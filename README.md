# 我的餐廳清單
![Restaurant List print screen](./public/image/PrintScreen.png)

## 介紹
紀錄屬於自己的餐廳清單，可以瀏覽餐廳、查看詳細資訊、甚至連結到地圖。

### 功能
- 查看所有餐廳清單
- 依關鍵字搜尋特定餐廳
- 瀏覽餐廳的詳細資訊
- 連結餐廳的地址至Google Map
